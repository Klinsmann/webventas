¿Que voy aprender?

Parte 1
1
Módulo

En el primer módulo vas a conocer las bases del desarrollo web, todo lo que debes saber antes de empezar a codear código HTML, CSS y Javascript. Aprenderás como se compone un proyecto web, los requisitos mínimos y necesarios para plantearte un desarrollo con los clientes que ya tienes además de todos fundamentos esenciales para crear sitios web exitosos.

2
Módulo

En este segundo módulo vas a crear YA tu primer página web de manera muy práctica, paso a paso y de forma muy estructurada usando HTML5 y sus etiquetas semánticas. Vamos a partir de una plantilla hecha en Photoshop para un cliente real de una florería donde vamos darle vida al diseño y aprenderemos a estructurar, seccionar, exportar y optimizar imágenes en Photoshop.

3
Módulo

¿Galerías, efectos, animaciones y transiciones? Claro que si! En este módulo vamos a ver desde el principio hasta cosas muy interesantes y avanzadas usando jQuery, el frameworks favorito y más usado de javascript. Un tema que debes de tener en tu arsenal de conocimientos si quieres llamarte a ti mismo “Ninja Front-End”.

4
Módulo

En este 4to modulo vas aprender a usar un framework css llamado “960 Grid system” una manera muy fácil y practica de hacer sitios web a partir de una serie de reglas css donde nos van a permitir hacer en cuestión de minutos una estructura de una página web bien bien simple, tienes que verlo para creerlo.

5
Módulo

Pero 960GS no es el único frameworks de css que veras, si no también veras como usar el framework de twitter “Bootstrap” otra opción muy recomendable de usar a la hora de desarrollar proyectos web en tiempo record, además veras como crear bocetos simples e interactivos con este framework de css en su version 3.

6
Módulo

Y por si fuera poco algo que es muy importante en el mundo es hacer que tus sitios web sean adaptados a los diferentes dispositivos, en este módulo realizaremos un proyecto completo con el método responsive web design. Si dominas bien este tema tendrás una amplia cartera de clientes que andan solicitando sus páginas web sean vistos por los diferentes dispositivos móviles.

Parte 2

Aunque es cierto que con estos 6 módulos estas más que preparado para ser todo un profesional en el mundo web, aún más contenido que debes aprender. Es por eso que si te registros HOY mismo a este entrenamiento obtienes 3 módulos extras que te ayudaran a expandir tus conocimientos.

7
Módulo

Aquí vas aprender a usar librerías que te ayudaran con las incompatibilidades que tiene internet explorer con HTML5, como quitar los prefijos de CSS3, Crear presentaciones atractivas, y cómo hacer que CSS se vuelva más como un lenguaje de programación.

8
Módulo

Crear un sitio web es apenas una fracción de todo el proceso completo para lograr el éxito en Internet. Una vez que aprendemos a crear nuestras páginas de manera profesional es hora de hacer que reciba muchos visitantes. Y una de las mejoras formas es a través de los motores de búsqueda como Google, Yahoo y otros...

9
Módulo

¿No sabes crear un plan de trabajo sobre tus proyectos web? No te preocupes...en este módulo veras todo el proceso, los días, como tener un calendario y saber decirle con certeza a tu cliente cuando tendrás listo y preparado su sitio web, además de crear diagramación para esto sitios.
